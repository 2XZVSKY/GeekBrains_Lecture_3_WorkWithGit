# Инструкция для работы с - MarkDown

## Выделение текста / -

Чтобы выделить текст *{Курсивом}* необходимо обрамить
его звездочкой
Например: (* *) [ *Курсив* ]

Чтобы выделить текст *{Жирным}* необходимо обрамить
его 2-умя звездочками
Например: (** **) [ **Жирный** ]
